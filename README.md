# MyNotes_NodeJS

MyNotes is a Node.js project where users can manage their notes. It provides functionalities to view, add, update, and delete notes.

## Features

- View all available notes
- Add new notes
- Udpdate existing notes
- Delete notes

## Technologies Used

- Node.js
- Express.js
- MongoDB
- HTML
- JavaScript
